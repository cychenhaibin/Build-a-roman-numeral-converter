# Build A Roman Numeral Converter

Convert Integers to Roman Numerals

## UX

**Getting Started**

Enter a number between 1 and 3999 in the input field.  Click on the convert button.  You will see either the roman numeral or error message.

Press the enter key to clear the input.

**User Stories**

1. As a user, I should have an `input` element with `id` of `number`.

2. As a user, I should have a `button` element with `id` of `convert-btn`.

3. As a user, I should have a `div` element with `id` of `output`.

4. As a user, when I click on  on the `#convert-btn` element without entering a value into the `#number` element, the `#output` element should contain the text `Please enter a valid number`.

5. As a user, if I enter the number `-1` in the `#number` element and the `#convert-btn` element is clicked, the `#output` element should contain the text `Please enter a number greater than or equal to 1`.

6. As a user, if I enter the number `4000` or greater in the `#number` element and the `#convert-btn` element is clicked, the `#output` element should contain the text `Please enter a number less than or equal to 3999`.

7. As a user, if I enter `9` in the `#number` element and the `#convert-btn` element is clicked, the `#output` element should contain the text `IX`.

8. As a user, if I enter `16` in the `#number` element and the `#convert-btn` element is clicked, the `#output` element should contain the text `XVI`.

9. As a user, if I enter `649` in the `#number` element and the `#convert-btn` element is clicked, the `#output` element should contain the text `DCXLIX`.

10. As a user, if I enter `1023` in the `#number` element and the `#convert-btn` element is clicked, the `#output` element should contain the text `MXXIII`.

11. As a user, if I enter `3999` in the `#number` element and the `#convert-btn` element is clicked, the `#output` element should contain the text `MMMCMXCIX`.

## Features

Allows the user to enter an integer in order to get the roman numeral.

Uses HTML5, CSS3 and JavaScript.
